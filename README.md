Snake Game CLI - Play in Your Terminal

A simple snake game for your command line. No fancy graphics, just pure text-based fun.

How to Play:
- Download and run the game with Python
- Use arrow keys or WASD to move your snake
- Eat the food to grow bigger and get points
- Don't hit the walls or yourself
- Press Q to quit anytime

What You Need:
- Python 3 (any version will work)
- A terminal or command prompt
- That's it! No internet needed

Why This Game:
- Works offline completely
- Super easy to run
- Fun way to kill some time
- Good for learning Python code

Just download it, run it, and start playing. No setup hassles.
